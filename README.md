# A simple joy to key python implementation

# How it works / Aim
The joy_to_key script takes in two json files. One that shows how the relevant python package 'understands' your controller inputs, `config.json`, and one that maps that 'understanding' to actual keyboard keys, `keyboard_map.json`

# Where to start
First run the `config.py` script to see how the relevant python package 'understands' your controller inputs. That will help you in creating you `config.json` file.  
Second you'll have to find what keyboard keys to map to your config.json keys
